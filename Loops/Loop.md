## While Loop

Syntax:

```python
while (condition):
    # code to be executed
```

Example:

```python
counter = 0
while counter < 5:
    print("The counter is:", counter)
    counter = counter + 1
```

## For Loop

```python
fruits = ["apple", "banana", "orange"]
for item in fruits:
    print(item)
```

```python
for i in range(0, 5):
    print("Python")
```
